Detailed Description

The DataManager is used to associate basic datatypes to an entity.

By sending a message to the data manager you can add one or more data entries to an entity. Each entry contains one DataBuffer. In the beginning of the DataBuffer the EntryHeader is located(which contains the Value for each DataType entry. The strings are stored at the end of the DataBuffer.
